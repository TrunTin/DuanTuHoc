--- Block, inline, and inline-block explained ---
- Các thẻ HTML đều có thuộc tính display: block/inline/inline-block

+ Block: luôn bắt đầu ở dòng mới, chiếm 100% theo width của thẻ cha, height tự co theo nội dung(content)
 trình duyệt tự động thêm
margin trước và sau phần tử.

+ Inline: Hộp có kích thước phụ thuộc vào nội dung bên trong nó
*Lưu ý với thẻ Inline: Có 1 số thuộc tính CSS không áp dụng được VD:
padding-top, padding-bottom, margin-top, margin-bottom, width, height, min-width, min-height
**Giải thích: Bởi vì thẻ inline được thiết kế để nằm trên cùng 1 hàng và không phá vỡ cấu trúc dòng. 

+ Inline-block: Sự kết hợp của 2 khối inline và block

--- Cú pháp sử dụng transition ---
+ C1: transition: <property>/all <duration> <timing-function> <delay>
    - property: thuộc tính css áp dụng
    - duration: thời gian hiệu lực
    - timing-function: xác định tốc độ thay đổi trong suốt quá trình:
        + ease -> mặc định: chậm đầu, nhanh giữa, chậm cuối
        + linear -> đều từ đầu đến cuối
        + ease-in -> chầm đầu, nhanh cuối
        + ease-out -> nhanh đầu, chậm cuối
        + ease-in-out -> chầm đầu, nhanh giữa, chậm cuối
        + cubic-bezier(z1, y1, x2, y2) -> tự định nghĩa đường cong
    - delay: thời gian bắt đầu thực hiện

--- CSS Units - Đơn vị trong CSS ---
- Đơn vị tuyệt đối - absolute lengths:
    + px(pixels) - pictrue elements - điểm ảnh: Là đơn vị cơ bản để đo kích thước
    1 hình ảnh kỹ thuật số

- Đơn vị tương đối - Relative lengths:
    + %: Kịch thước bằng n% so với phần tử cha
    + vw: n% chiều rộng khung hình (viewport - phần hiển thị trên browser)
    + vh: n% chiều cao khung hình (viewport - phần hiển thị trên browser)
    + em: Kích thước gấp n.em lần font size hiện tại (có kế thừa)
    + rem: kích thước gấp n.rem lần font size root element - html mặc định font-size 16px (Không kế thừa),

- Đổi px sang rem, mặc định font-size html 16px, 1rem = 16px ==> 10px = 0.625 rem 
hoặc đặt html {
    font-size: 62.5% (16px * 62.5% = 10px)
}
==> 1rem = 10px vì nó phụ thuộc vào kích thước font-size root (html)
Ví dụ: 220px => 22rem
